# VS Code Keyboard Shortcuts - Mac

| Action | Shortcut |
|---|---|
| **Single line comment** | `Cmd + /` |
| **Block comment** | `Shift + Option + A` |
| **Open terminal** | `Ctrl + `` ` `` |
| **Save file** | `Cmd + S` |
| **Save all files** | `Cmd + Option + S` |
| **Undo** | `Cmd + Z` |
| **Redo** | `Cmd + Shift + Z` |
| **Copy line down** | `Shift + Option + ↓` |
| **Copy line up** | `Shift + Option + ↑` |
| **Move line down** | `Option + ↓` |
| **Move line up** | `Option + ↑` |
| **Delete line** | `Cmd + Shift + K` |
| **Select entire line** | `Cmd + L` |
| **Multi-cursor** | `Option + Click` |
| **Select next occurrence** | `Cmd + D` |
| **Format document** | `Shift + Option + F` |
| **Go to line** | `Ctrl + G` |
| **Quick open file** | `Cmd + P` |
| **Command palette** | `Cmd + Shift + P` |
| **Split editor** | `Cmd + \` |
| **Toggle sidebar** | `Cmd + B` |
| **Zoom in** | `Cmd + =` |
| **Zoom out** | `Cmd + -` |
| **Find** | `Cmd + F` |
| **Find & Replace** | `Cmd + H` |
